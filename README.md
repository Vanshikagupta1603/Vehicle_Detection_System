🚗 Vehicle Detection & Traffic Density Analysis System

📌 Overview
This project is an AI-powered traffic monitoring system developed using YOLOv8, OpenCV, Streamlit, Pandas, and Matplotlib. The system detects vehicles from images and videos, analyzes traffic density, generates CSV reports, and visualizes traffic statistics through interactive charts.

✨ Features
Vehicle Detection using YOLOv8
Supports both Images and Videos
